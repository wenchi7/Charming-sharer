# Charming-Sharer

![Website Screenshot](src/assets/web-snapshot.png)

<br>
因為喜歡美妝，所以選擇打造一個美妝分享的平台。
在資訊爆炸的時代，希望美妝愛好者有一個可以輕鬆找到自己想要什麼的空間。
風格樣式以美式復古色系為主軸，參考美妝品牌ColourPop(https://colourpop.com)。
